Pousse-Café Framework

Making Domain-Driven Design implementation easy

Quick description

Pousse-Café is a Java framework that eases the writing of applications following Domain-Driven Design (DDD) methodology. While DDD has proven to be useful in implementing complex business rules and/or fulfilling strong scalability requirements, it is not always easy to apply given the number of surrounding subjects that need to be addressed (uncoupling of domain logic and storage engine, robust handling of Domain Events, testing, etc.). Pousse-Café's purpose is to solve most of those issues allowing developers to focus on the Domain logic and its implementation.
