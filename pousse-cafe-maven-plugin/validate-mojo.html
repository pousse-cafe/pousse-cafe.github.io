pousse-cafe:validate

Full name:

org.pousse-cafe-framework:pousse-cafe-maven-plugin:0.19.0:validate

Description:

Validates the project source code.

Error or warning messages are generated. The build fails if errors are detected.

Attributes:

  • Requires a Maven project to be executed.
  • Requires dependency resolution of artifacts in scope: compile+runtime.

Optional Parameters

Name Type Since Description
<basePackages> String[] 0.19 Base packages used for classpath exploration. No base package implies no classpath exploration (default behavior).
User property is: basePackages.
<failOnWarn> boolean 0.19 If true, makes the build fail with warnings. False by default.
Default value is: false.
